Project: "Aggie Platter"

May 2023 - June 2023

Link to Github here


I worked with a team to create a program to help users track nutrient intake at UC Davis dining commons. This program uses the BeautifulSoup and Requests python libraries to web scrape the dining commons menus. The program takes in user input and outputs the number of calories the user consumed, as well as the number of grams of fat, carbohydrates, and protein. Play the video for a demo of the program.
